# Icono de Project-Cronia

Para cambiar el icono de la app:

1. Coloca aquí una imagen PNG cuadrada, preferiblemente de 1024x1024 px.
2. Nómbrala `app_icon.png`.
3. Agrega al final de `pubspec.yaml` la configuración indicada en `PROJECT_CRONIA_NOTES.md`.
4. Ejecuta:

```bash
dart run flutter_launcher_icons
flutter clean
flutter pub get
flutter run
```
